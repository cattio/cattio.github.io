about fashion cay

Our website was made as the final project for Girls Who Code Software.org/ Georgetown Law Summer Immersion Program in August 2019. The name "Phashion Cay" comes from the word fashion and our names; Catherine, Alaysha, and Yeira. For our website, we used HTML, CSS, and Javascript, and hosted it using GitHub pages. We used W3Schools a lot for coding reference. We used Trello for project management. The website was made without a template and uses the "Great Vibes" font from google fonts for headings, and Verdana font for text. The style quiz and size calculators both use Javascript and were made without templates. The store options map uses an embedded custom Google map. Size information and pictures are from the H&M website sizeguide.
